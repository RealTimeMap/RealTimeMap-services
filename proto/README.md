
# Описание и назначение
## Исходники proto-файлов, единый источник правды для API
### Папка служит для хранения .proto файлов на осное которых генерируется код для gRPC


## Архитектура
 - Название папки должна быть понятна для группировки .proto файлов. Пример:
```
|-proto
    |- user # все .proto файлы для работы с user (GetById, GetByIds, GetActiveBan и тд) 
    |- gamefication # Все для работы с Геймификацией 
```
